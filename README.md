# Sadhana Shree - Portfolio

A modern, responsive developer portfolio showcasing projects, skills, and certifications.

## 🚀 Features

- **Modern UI/UX**: Clean, professional design with dark/light mode support.
- **Responsive Design**: Optimized for all devices, from mobile to desktop.
- **Interactive Elements**: Smooth animations with Framer Motion and 3D effects with Three.js.
- **Performance**: Built with Vite and React for lightning-fast loading speeds.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Animations**: Framer Motion, Three.js
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Firebase

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
